# Music Big Data Analytics

## Dataset Setup

This project requires two datasets that must be downloaded and placed in the project root directory:

1. **dataset.csv** - Spotify Tracks Dataset  
   Download from: https://www.kaggle.com/datasets/maharshipandya/-spotify-tracks-dataset

2. **dataset2.csv** - Records: A Comprehensive Music Metadata Dataset  
   Download from: https://www.kaggle.com/datasets/fleshmetal/records-a-comprehensive-music-metadata-dataset

**Important:** The datasets must be named exactly as `dataset.csv` and `dataset2.csv` for the notebook to run correctly.
